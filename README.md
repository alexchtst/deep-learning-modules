# Deep Learning Modules

This project is a repository to create Deep Learning Module such as Conv2d, Neural, Flatten, Pooling and Conv2dTranspose Layer to build deep learning model. The custom model built to enhance the algorithm and research to train model and implement in fpga or other hardware or implement it to other programing language.
